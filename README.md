# easy-lowcode

##  🔥 最简单的低代码实现

技术栈： React18 + Vite + codeMirror + Koa2

1. 实现拖拽生成组件代码的json描述的页面 √
2. 物料区 √
3. 编辑器 √
4. 通信 √
5. 生成不同框架源代码(出码) √
6. 服务端koa2构建生成源码 √
7. H5移动端生成适配 √
8. 支持sdk加载页面，renderer解析 √
9. iframe隔离 postMessage 通信  √

12. canvas编辑低代码组件 ×
